tabHeader
功能
  • tab标签添加
  • tab标签切换
  • tab标签关闭
  • tab多出实现滑动效果
  • 支持搜索
参数
参数 参数类型 是否必须 可选值 默认值 说明
element jQuery Object 是 合法的jQuery对象 标签栏
defaultTab Object 合法的对象 默认生成标签栏时，添加的第一个标签。
search Object 合法的search参数对象 显示搜索框，参考search组件参数data,placeholder
onSelect Function 可执行的函数 选择某标签时触发的函数，参数data。
onClose Function 可执行的函数 关闭某标签时触发的函数,参数data。
onAddTab Function 可执行的函数 添加某标签时触发的函数,参数data。
方法
方法名 参数说明 返回值 说明
addTab {label: '标签名', id: 1} 添加标签，先触发onAddTab，再触发onSelect
示例
简单的标签栏
var tab1 = new $.jqcTabHeader({
    element: $('#div1'),
    defaultTab: {
        label: 'test1',
        id: 1
    }
});
$('#btn1').click(function () {
    var _id = Math.random() * 10000000 | 0;
    tab1.addTab({
        label: _id.toString(16),
        id: _id
    });
});
带搜索功能的标签栏
var tab2 = new $.jqcTabHeader({
    element: $('#div2'),
    defaultTab: {

$('#btn3').click(function () {

var $tabContainer = $('#tab_container');

var tab3 = new $.jqcTabHeader({
    element: $('#div3'),
    defaultTab: {
        label: 'test1',
        id: '1'
    },
    onAddTab: function (data) {
        var _container = $('<div>')
            .attr('data-id', data.id)
            .text(data.label);
        $tabContainer.append(_container);
    },
    onSelect: function (data) {
        $tabContainer.find('[data-id='+data.id+']')
            .show()
            .siblings()
            .hide();
    },
    onClose: function (data) {
        $tabContainer.find('[data-id='+data.id+']')
            .remove();
    }
});

$('#list li').click(function () {
    var _label = $(this).text();
    var _id = $(this).attr('data-uid');
    tab3.addTab({
        label: _label,
        id: _id
    });
});
