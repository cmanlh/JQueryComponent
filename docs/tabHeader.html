tabHeader
功能
  • tab标签添加
  • tab标签切换
  • tab标签关闭
  • tab多出实现滑动效果
  • 支持搜索
参数
参数 参数类型 是否必须 可选值 默认值 说明
data JSON Array,
JSON Object		 是 合法的数据类型 菜单数据源
img String 合法的图片路径 菜单顶部图片
speed Number 正整数 200 菜单动画过渡时长，单位毫秒
show Booble false 组件生成时是否显示菜单
defaultLabel String data对象内label属性值 组件生成时的默认值，具有选中效果。
onSelect function 可执行的函数 菜单选中后执行的回调，参数为被选中的data
方法
方法名 参数说明 返回值 说明
showMenu 划出菜单
hideMenu 收起菜单
setCurrentLabel 传入需要选中菜单对应的label 菜单自动展开并定位到该选项，并触发onSelect事件。

基本菜单栏
var menu;

$('#btn_create').one('click', function () {
    menu = new $.jqcMenu({
        data: data,
        onSelect: function (data) {
            $('#div1').text(JSON.stringify(data));
        }
    });
});

$('#btn_show').click(function () {
    if (menu) {
        menu.showMenu();
    }
});
$('#btn_hide').click(function () {
    if (menu) {
        menu.hideMenu();
    }
});
$('#btn_select').click(function () {
    if (menu) {
        menu.showMenu();
        menu.setCurrentLabel('三级菜单03-01-02');
    }
});
