formToolBar
功能
  • 表单搜索框
  • 多行可展开，可收起
  • 节约页面视图空间
参数
参数 参数类型 是否必须 可选值 默认值 说明
element jQuery Object 是 组件container
conditionHtml Html 条件区域（左侧）的html代码片段
controlHtml Html 控制区域（右侧）的html代码片段
height Number(int) 36 组件高度，超出该高度实现折叠功能，单位：px
background String ‘#ffffff’ 组件背景颜色
afterRender Function 组件渲染之后的回调函数
noResize Function（height） 重新设置尺寸后的回调
onChange Function(height) 展开收起回调
方法
方法名 参数说明 返回值 效果及作用
fold 折叠组件
spread 展开组件
resize 组件内视图改变后调用该方法，确定是否需要折叠及显示‘展开按钮’
示例
正常示例
我是bar下面的内容
var bar1 = new $.jqcFormToolBar({
    element: $('#div1'),
    conditionHtml: '<div class="condition"><label><span>姓名：</span><input type="text"/></label><label><span>姓名：</span><input type="text"/></label><label><span>姓名：</span><input type="text"/></label><label><span>姓名：</span><input type="text"/></label><label><span>姓名：</span><input type="text"/></label><label><span>姓名：</span><input type="text"/></label><label><span>姓名：</span><input type="text"/></label><label><span>姓名：</span><input type="text"/></label></div>',
    controlHtml: '<div><button>测试1</button><button>测试2</button><button>测试3</button></div>'
});

$('.btn1').click(function () {
    bar1.spread();
});

$('.btn2').click(function () {
    bar1.fold();
});

$('.btn3').click(function () {
    var _height = $('.condition').height();
    _height = _height === 36 ? 'auto' : 36;
    $('.condition').height(_height);
    bar1.resize();
});
