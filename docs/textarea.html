textarea
功能
  • 拓展原生textarea
  • 自适应铺满屏幕
  • 增加可输入量
参数
参数 参数类型 是否必须 可选值 默认值 说明
el jQuery Object（input,textarea） yes 需要拓展的input或textarea对象
title String 编辑 弹框title
placeholder String 请输入内容... placeholder
cancelBtnText String 取消 取消按钮文案
saveBtnText String 保存 保存按钮文案
closeByModal Boolean false 是否可以通过点击modal关闭输入框
beforeSave Function 保存之前回调，参数(value, next)next表示下一步即保存并关闭输入框
示例
正常示例
new $.jqcTextarea({
    el: $('#textarea1')
});
异步保存
var loading = new $.jqcLoading();

new $.jqcTextarea({
    el: $('#textarea2'),
    beforeSave: function (value, next) {
        loading.show('保存中，2秒后关闭...');
        setTimeout(() => {
            loading.hide();
            next();
        }, 2000);
    }
});
