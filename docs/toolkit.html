toolkit
功能
  • 为全局添加'$.jqcToolkit'工具库
方法
方法名 参数说明 返回值 效果及作用
uuid Number(int)默认为8 16进制随机数字符串 传入一个正整数，返回该长度的16进制随机字符串。
rawType Any 数据类型 传入一个数据，返回该数据的原生类型如：Array Object Function ...。
示例
uuid

$.jqcToolkit.uuid();    // '0a5cc415'

$.jqcToolkit.uuid(6);    // 'b457da'
rawType

$.jqcToolkit.rawType([]);    // 'Array'

$.jqcToolkit.rawType({});    // 'Object'
