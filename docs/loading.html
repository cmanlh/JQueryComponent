loading
功能
  • loading效果
  • 防滚动
  • 防点击
  • 可设置大小，速度
参数
参数 参数类型 是否必须 可选值 默认值 说明
width Number(int) 300 loading宽度
height Number(int) 150 loading高度
show Boolean false 创建之后是否自动显示
speed Number(int) 1-10 1 loading速度，1最慢，10最快
方法
方法名 参数说明 返回值 效果及作用
show 显示loading
hide 隐藏loading
示例
默认loading展示
var loading = new $.jqcLoading();

$('#btn1').click(function () {
    loading.show();
    setTimeout(function () {
        loading.hide();
    }, 5000);
});
