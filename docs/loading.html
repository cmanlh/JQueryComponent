loading
功能
  • loading效果
  • 防滚动
  • 防点击
  • 可设置大小，速度
方法
方法名 参数说明 返回值 效果及作用
show 需要显示的文字 显示loading，及文字
hide 隐藏loading
lock 毫秒数，0表示永久锁定 锁定当前loading状态
unlock 解锁当前loading状态
示例
默认loading展示
var loading = new $.jqcLoading();

$('#btn1').click(function () {
    loading.show('我5秒后消失');
    setTimeout(function () {
        loading.hide();
    }, 5000);
});
