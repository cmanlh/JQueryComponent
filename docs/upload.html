upload
功能
  • 分为单文件，多文件模式
  • 可限制文件大小
  • 可自定义文件类型
  • 可提交自定义数据
  • 可自定义文件name字段
参数
参数 参数类型 是否必须 可选值 默认值 说明
el jQury Object yes 容器元素
url String yes 上传文件的地址
data Object|Function 需要额外上传的formData
name String file 文件对应的name
accept Array(String) 文件格式|image|word|excel 可选文件类型，一种文件类型可直接用字符串，如：'jpg',多种文件用数组，如:['jpg', 'pdf'],image表示所有图片类型，word表示新老word文档类型，excel表新老表格类型。
mode String single|multiple single 单文件，多文件模式
placeholder String placeholder提示
maxSize Int 0 限制文件大小，单位KB,0标识不限制
uploadBtnText String 上传数据 上传按钮的文本
selectFilesText String 选取文件 选择文件按钮的文本
success Function（data, successCallback, errorCallback） 请求成功的回调函数，data为接口返回的数据，successCallback自定义成功的回调,调用时需要传入map对象，map对象已{前端uid: 后台返回uid}形式传入，errorCallback自定义失败的回调
check Function(data) 返回值为Boolean, true执行下一步，false中断上传
onRemove Function(uid, successCallback, errorCallback) 监听删除文件操作，回调函数可传字符串参数作为提醒文字，若不传则不提醒
示例
单文件模式
new $.jqcUpload({
    el: $('.single'),
    url: 'http://172.29.114.86:7001/upload',
    accept: 'image',
    maxSize: 800,
    name: 'test',
    data: {
        username: 'mawenjie',
        password: '111111'
    },
    placeholder: 'placeholder'
});
多文件模式
new $.jqcUpload({
    el: $('.multiple'),
    url: 'http://172.29.114.86:7001/upload',
    accept: ['xlsx', 'txt', 'jpg'],
    mode: 'multiple',
    placeholder: '请选择xlsx、txt、jpg格式文件,我是placeholder'
});
