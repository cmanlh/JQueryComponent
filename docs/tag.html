tag
功能
  • 标签组件
  • 可自定义添加
  • 可选择
  • 只读
参数
参数 参数类型 是否必须 可选值 默认值 说明
el jQuery Object yes 需要添加通知的页面input元素，有disabled属性时只读
width Int 400 组件宽度
data Array 标签备选项
adapter Object {value: 'value',label: 'label'}} 适配器
defaultValue String 默认值
示例
正常示例
new $.jqcTag({
    el: $('#tag1')
});
标签备选项
new $.jqcTag({
    el: $('#tag2'),
    data: [{
        id: 1,
        text: '简单'
    }, {
        id: 2,
        text: '普通'            
    }, {
        id: 3,
        text: '困难'            
    }, {
        id: 4,
        text: '深渊'            
    }],
    adapter: {
        value: 'text',
        label: 'text'
    }
});
默认值
new $.jqcTag({
    el: $('#tag3'),
    defaultValue: '新手;顽强'
});
