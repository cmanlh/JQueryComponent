select
功能
  • 模拟浏览器原生下拉框
  • 支持默认值
  • 支持全部和空值拓展
  • 支持格式化
  • 支持适配
  • 支持原生change事件监听
参数
参数 参数类型 是否必须 可选值 默认值 说明
el jQuery Object 是 需要组件化的input标签-jQuery对象
defaultValue String 默认值
data Array [] 数据源
adapter Object {value: 'value', label: 'label'} 字段适配，label可以为function
onSelect Function 选择选项的回调函数args:(data)，也可以监听input的change事件args:（event， data）
示例代码
标准数据模型，无需适配参数
const normalData = [{
    value: 0,
    label: '广兰路'
}, {
    value: 1,
    label: '金科路'
}];

new $.jqcSelect({
    el: $('#normal'),
    data: normalData
});
拓展全部和空值
// 全部和空值的值都是空字符串
<input id="all" ext="*"/>
new $.jqcSelect({
    el: $('#all'),
    data: normalData
});

<input id="empty" ext="none"/>
new $.jqcSelect({
    el: $('#empty'),
    data: normalData
});
默认值
// 行间属性的默认值优先级更高
<input id="default1" ext="*"/>
new $.jqcSelect({
    el: $('#default1'),
    data: normalData,
    defaultValue: 1
});

<input id="default2" ext="*" defaultValue=""/>
new $.jqcSelect({
    el: $('#default2'),
    data: normalData,
    defaultValue: 1
});
适配

const adapterData = [{
    id: 1,
    name: '广兰路'
}, {
    id: 2,
    name: '金科路'
}];
new $.jqcSelect({
    el: $('#adapter'),
    data: adapterData,
    adapter: {
        value: 'id',
        label: 'name'
    },
    onSelect: function (data) {
        console.log(data);
    }
});
