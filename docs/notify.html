notify
功能
  • 通知模块
  • 超出最大高度可滚动
  • 可监听开关事件
参数
参数 参数类型 是否必须 可选值 默认值 说明
el jQuery Object yes 需要添加通知的页面元素
data Array 通知消息列表
position String 'left,right' left 通知框显示的位置
adapter Object {title: 'title',content: 'content', time: function(data){return new Date();}} 适配器
tip Function 标题后面的提示文字
onOpen Function 监听通知列表打开事件
onClose Function 监听通知列表关闭事件
onMore Function 监听点击title事件，不设置即title不可点击
onSelect Function 监听点击单个通知事件，不设置即通知不可点击
示例
正常示例
new $.jqcNotify({
    el: $('.notify1')
});
带有适配数据
new $.jqcNotify({
    el: $('.notify2'),
    data: [{
        title: '抬头1',
        content: '内容内容内容内容内容内容内容内容内容内容内容内容内容内容内容内容内容',
        createTime: 1542009959382
    }],
    adapter: {
        time: function (data) {
            return new Date(data.createTime);
        }
    },
    tip: function (data) {
        return `${data.length}条未读`;
    }
});
