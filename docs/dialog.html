Dialog
功能
  • 弹出框
参数
参数 参数类型 是否必须 可选值 默认值 说明
title String 是 文本或者html片段 弹出框标题
content String 是 文本或者html片段 弹出框内容
modal boolean true 该弹窗是否为模态窗口，模态窗口带有屏蔽层。
width int 正整数 600 弹出框的默认像素宽度
beforeClose function 回调函数 弹出框关闭前回调
beforeOpen function 回调函数 弹出框打开前回调
afterOpen function 回调函数 弹出框打开后回调
afterClose function 回调函数 弹出框关闭后回调
示例
弹出框示例
                    $('#basic').on('click', function (e) {
                        var dialog = new $.jqcDialog({
                            title: 'hello world',
                            content: '<p><h1>Hello world</h1></p>'
                        });
                        dialog.open();
                    });
