nav
功能
  • 多层级按钮导航
参数
参数 参数类型 是否必须 可选值 默认值 说明
el jQuery Object yes 承载导航的容器
data Array [] 按钮数据
adapter Object {value: 'value', children: 'children', disabled: 'disabled'}} 适配器
onSelect Function 点击事件回调
示例
var btns = [{
    label: '一级菜单-1',
    children: [{
        label: '二级菜单-1'
    }, {
        label: '二级菜单-特别长特别长特别长特别长',
        children: [{
            label: '三级菜单-1'
        }, {
            label: '三级菜单-特别长特别长特别长特别长'
        }, {
            label: '三级菜单-3',
            disabled: true
        }, {
            label: '三级菜单-4'
        }]
    }, {
        label: '二级菜单-3',
        disabled: true
    }, {
        label: '二级菜单-4'
    }]
}, {
    label: '一级菜单-2'
}, {
    label: '一级菜单-3',
    disabled: true
}, {

    }
