menuTree
功能
  • 菜单树组件
  • 支持三级菜单
  • 可配置菜单选项
  • 支持独立选项自动越级
  • 自动收起
  • toggle触发收起，展示
参数
参数 参数类型 是否必须 可选值 默认值 说明
data Array Object 是 该参数为菜单的数据
adapter Object {id:'id', label:'label', child:'child'} 适配data数据格式
autoSkip Boolean true 单菜单是否自动越级
displayed Boolean true 菜单渲染后是否立即显示
allowedConfig Boolean false 是否允许配置菜单，需配合configurableMenuData一起使用
configurableMenuData Array Object 可配置的菜单选项数据源，格式同data,且包含data。
configBoxWidth Number(int) 500 配置框宽度
speed Number(int) 200 菜单滑入，划出动画时长，单位:ms
width Number(int) 150 菜单宽度，单位:px
position String 'static','relative','absolute','fixed' 'fixed' 菜单定位方式，同CSS定位方式值
top Number(int) 0 菜单定位top值
left Number(int) 0 菜单定位left值
autoHide Boolean true 鼠标移除菜单是否自动收起
trigger $(selector) 触发收起，展示的jquery对象
triggerFirst Boolean false 菜单创建完成是否默认触发第一个可用菜单选项
onSelect Function 选中菜单选项触发的回调函数，回调参数为对应的data数据
onShow Function 选中菜单展示时触发的回调（width,speed）
onHide Function 选中菜单收起时触发的回调（width,speed）
方法
方法名 参数说明 返回值 效果及作用
show 展示菜单
hide 隐藏菜单
示例
标准数据模型，无需适配参数。
var basicData = [{
    id: '1',
    label: '一级菜单01',
}, {
    id: '2',
    label: '一级菜单02',
}, {
    id: '3',
    label: '一级菜单03',
}, {
    id: '4',
    label: '一级菜单04',
}];

var menu1;
$('.basic_btn_1').one('click', function () {
    menu1 = new $.jqcMenuTree({
        data: basicData
    });
});
$('.basic_btn_2').click(function () {
    menu1 && menu1.hide();
});
$('.basic_btn_3').click(function () {
    menu1 && menu1.show();
});
非标准数据模型，使用适配参数
var atypicalData = [{
    uid: '1',
    name: '菜单二01',
}, {
    uid: '2',
    name: '菜单二02',
}, {
    uid: '3',
    name: '菜单二03',
}, {
    uid: '4',
    name: '菜单二04',
}];

$('.atypical_btn_1').one('click', function () {
    var menu2 = new $.jqcMenuTree({
        data: atypicalData,
        adapter: {
            id: 'uid',
            label: 'name',
            child: 'child'
        },
        left: 200
    });
});
带有子菜单的菜单栏，及不自动跳级菜单
var data2 = [{
    id: 1,
    label: '菜单三01',
    child: [{
        id: 11,
        label: '菜单三01-01',
        child: [{
            id: 111,
            label: '菜单三01-01-01'
        }]
    }]
}];

$('.default_btn').one('click', function () {
    new $.jqcMenuTree({
        data: data2,
        left: 500
    })
});
$('.noskip_btn').one('click', function () {
    new $.jqcMenuTree({
        data: data2,
        left: 700,
        autoSkip: false,
    })
});
