calendar
功能
  • 日历
  • 可编辑
参数
参数 参数类型 是否必须 可选值 默认值 说明
el jQury Object yes 日历容器元素
title String|Function 日历title
cellHeight Int 100 日历cell高度
startYear Int now - 10 开始年份
endYear Int now + 10 结束年份
data Array 特殊日期
adapter Object(content, startTime, endTime, tip, type) 适配data
currentYear Int now 默认的年份
currentMonth Int 1-12 now 默认月份
cellTextAlign String left|center|right center cell文字位置
onSelect Function 选中cell触发的事件
cellRender Function(data, cell, memos) 渲染cell时回调，须返回memos
canEditor Boolean false 是否能够编辑
onMemoSelect Function（memo） 选中日程触发的回调
onMemoChange Function（memo） 日程列表改变时触发
onMemoAdd Function（memo） 新增日程触发
onMemoUpdate Function（memo） 更新日程触发
onMemoDelete Function（memo） 删除日程触发
onMonthChange Function（year, month） 月份变更触发
示例
new $.jqcCalendar({
    el: $('.calendar'),
    title: function (year, month) {
        return `${year}年${month}月`
    },
    data: [{
        date: '2018-11-11',
        text: '光棍节'
    }, {
        date: '2018-10-01',
        text: '国庆节'
    }],
    adapter: {
        content: 'text',
        startTime: function ({date}) {
            return +new Date(date);
        },
        endTime: function ({date}) {
            return +new Date(date);
        }
    }
});
// calendar.reRender可用于重新渲染日历，参数可传新的data
