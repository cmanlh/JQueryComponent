confirm
功能
  • 确认弹框
  • 单例模式,一次引入，全局使用
参数
参数 参数类型 是否必须 可选值 默认值 说明
title String 确认框title
content String|jQury Object 确认框内容
width Int 400 确认框宽度
onConfirm Function 点击确认触发
onCancel Function 点击取消触发
cancelText String 取消 取消按钮文字
confirmText String 确定 确定按钮文字
btnPosition String left|center|right right 按钮位置
icon String|false success|error|warn|info|false error 是否显示icon
示例
$('#btn1').click(function () {
    $.jqcConfirm({
        title: '我是title',
        content: '我是内容',
        onConfirm: function () {
            console.log('confirm');
        },
        onCancel: function () {
            console.log('cancel');
        }
    });
});
