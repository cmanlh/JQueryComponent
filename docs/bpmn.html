Bpmn/BpmnViewer
功能
  • 流程图编辑/流程图查看
  • 支持*.bpmn导入
  • 支持导出*.bpmn文件
  • 支持导出*.svg文件
  • 复核bpmn2.0规范
提示
  • Ctr + 鼠标滚轮,可进行缩放操作
参数
参数 参数类型 是否必须 可选值 默认值 说明
container String/Element Object 是 容器，“.className #id”
方法
方法名 参数 是否必须 可选值 默认值 说明
importXML xml 是 bpmn文件内的文本
exportBpmn 导出Bpmn文件
exportSvg 导出svg文件
openFile 打开文件
示例代码
// 查看bpmn请调用$.jqcBpmnViewer
var bpmn = new $.jqcBpmn({
    container: '.bpmn_container'
});
$('.openFile').click(function () {
    bpmn.openFile();
});
$('.exportBpmn').click(function () {
    bpmn.exportBpmn();
});
$('.exportSvg').click(function () {
    bpmn.exportSvg();
});
