search
功能
  • 本地数据搜索；
  • 支持拼音首字母搜索；
参数
参数 参数类型 是否必须 可选值 默认值 说明
element jQuery Object 是 可选的jQuery对象 搜索框的容器，决定搜索框的大小。
placeholder String 合法的字符串 搜索框内的提示文字
data Array Object 是 合法的数组对象 搜索的数据
onSelect Function 可执行的函数 搜索后选中搜索结果执行的函数,参数为对应的data数据
示例
简单的数据格式
var simpleData = [{
    label: '测试1',
    id: 1
}, {
    label: '测试2',
    id: 2   
}, {
    label: '拼音3',
    id: 3   
}];

new $.jqcSearch({
    element: $('#div1'),
    placeholder: '我是搜索框1',
    data: simpleData,
    onSelect: function (data) {
        console.log(data);
    }
});
带有层级的数据
var data = [{
    label: '测试1',
    child: [{
        label: '测试1-1',
        id: 1
    }]
}, {
    label: '测试2',
    id: 2
}, {
    label: '拼音',
    child: [{
        label: '拼音1',
        id: 3
    }, {
        label: '拼音2',
        id: 4
    }]
}];

new $.jqcSearch({
    element: $('#div2'),
    placeholder: 'I am div2.',
    data: data,
    onSelect: function (data) {
        console.log(data);
    }
});
