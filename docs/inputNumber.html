InputNumber
功能
  • 自动格式化数值输入，支持中文，全角输入
参数
参数 参数类型 是否必须 可选值 默认值 说明
element jQuery Object 是 一个input[type=text] jQuery Object
decimals Integer 0 小数位数，不做四舍五入处理
示例
整数输入
new $.jqcInputNumber({
    element: $('input[name=basic]')
});
保留两位小数
new $.jqcInputNumber({
    element: $('input[name=withDecimals]'),
    decimals: 2
});
