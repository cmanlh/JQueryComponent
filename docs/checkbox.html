checkbox
功能
  • 支持单选模式、复选模式
  • 支持横向展示、竖向展示
参数
onChange: null
参数 参数类型 是否必须 可选值 默认值 说明
el $(INPUT) yes jquery对象（input）
type String checkbox|radio checkbox 样式类型
mode String single|multiple single 单选模式、复选模式
data Array(object) [] 数据
disabled String 置灰 多选如： 1,2,3 只读： *
defaultValue String 默认值
width String|Int 100% 宽度
direction String row|column row 展示方向
adapter Object {label:'label',value:'value'} 数据适配，label可以是function,参数1为对应数据，参数2为选中状态，返回字符串，或html片段
trigger String icon 是否仅icon触发选中
onChange Function 值变化后触发的回调
示例
默认值
    const dataSource = [{
            label: '测试1',
            value: 1
        }, {
            label: '测试2',
            value: 2
        }, {
            label: '测试3',
            value: 3
        }, {
            label: '测试4',
            value: 4
        }, {
            label: '测试5',
            value: 5
        }, {
            label: '测试6',
            value: 6
        }];

    new $.jqcCheckbox({
        el: $('#default'),
        data: dataSource,
        defaultValue: '4'
    });
复选
    new $.jqcCheckbox({
        el: $('#multiple'),
        data: dataSource,
        mode: 'multiple',
        defaultValue: '1,3,5'
    });
radio
    new $.jqcCheckbox({
        el: $('#radio'),
        data: dataSource,
        type: 'radio'
    });
纵向
    new $.jqcCheckbox({
        el: $('#column'),
        data: dataSource,
        direction: 'column'
    });
适配
    const dataSource1 = [{
        id: 0,
        name: '佩奇'
    }, {
        id: 1,
        name: '乔治'
    }];
    new $.jqcCheckbox({
        el: $('#adapter'),
        data: dataSource1,
        defaultValue: 0,
        direction: 'column',
        adapter: {
            label: function (data, isChecked) {
                return `我是小猪${data.name}${isChecked?',哼哼哼':''}！`
            },
            value: 'id'
        }
    });
