msg
功能
  • 提醒框
  • 多条可滚动
  • 可删除，可添加
参数
参数 参数类型 是否必须 可选值 默认值 说明
element jQuery Object 是 组件container
data Array Object 每条提示框内数据
width Number(int) 800 组件宽度，单位：px
speed Number(int) 1-10 1 组件滚动时的速度
adapter Object {text:'text'} 提示框内的文本适配data数据
click Function 点击触发事件
方法
方法名 参数说明 返回值 效果及作用
add Object || Array 添加数据，可单个添加，也可以多个一起添加
示例
正常示例
var msg = new $.jqcMsg({
    element: $('#div1'),
    data: [{
        label: '11111111',
    }, {
        label: '22222222',
    }],
    adapter: {
        text: 'label'
    },
    width: 500,
    click: function (data) {
        alert(data.label);
    }
});

$('.btn1').click(function () {
    var _val = $('#input1').val();
    if (_val) {
        msg.add({
            label: _val
        });
        $('#input1').val('');
    }
});

$('.btn2').click(function () {
    msg.add([{
        label: '多条1'
    }, {
        label: '多条2'
    }, {
        label: '多条3'
    }, {
        label: '多条4'
    }]);
});
